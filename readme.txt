GFG AM Presentation (Reveal.js)

How to open (Windows/Mac):
1) Open the 'index.html' file with Chrome or Edge.
2) Use Left/Right arrow keys to navigate the slides.
3) Press F11 for fullscreen.

How to export PDF (backup):
1) Open 'index.html' and append '?print-pdf' to the URL, e.g.:
   file:///.../presentation/index.html?print-pdf

3) Layout: Landscape, Margins: None or Minimum, Background graphics: Enabled.

Folder structure:
presentation/
  index.html
  /slides/  (11 slideX.html files)
  /assets/  (logos used by the slides)

Notes:
- Do not rename the files inside /assets/ or /slides/ unless you also update the HTML references.
- Reveal.js is loaded from a CDN, so internet connectivity helps for the first load; afterwards it may be cached.
