The Challenge We Had:

Excel and OneDrive were our "database", BUT this created serious limitations:

  • Data scattered across multiple Excel files
  • Manual consolidation required for reports
  • Limited to ~1M rows per file
  • Multiple file versions without proper tracking or governance
  • No data validation or relationship enforcement
  • Concurrent access issues when multiple users needed data

The Transformation:

Now powered by PostgreSQL - a professional-grade relational database:

  • Centralized data storage with unlimited scalability
  • Real-time access for all users simultaneously
  • Handles millions of records without performance bottlenecks
  • Automated data validation and integrity checks
  • Advanced querying capabilities for complex analysis
  • Direct connections to all our tools (n8n, Power BI, APIs)

Impact:

  • 100+ million rows managed efficiently
  • Sub-second query responses for critical reports
  • Zero data loss with automated daily backups
  • Single source of truth for all hotel data

Database Architecture:

  • raw schema: Direct ingestion from external sources
  • core schema: Cleaned and validated data
  • marts schema: Pre-aggregated data for dashboards
  • audit schema: Complete activity logging for compliance

The Result:

Professional data infrastructure that scales with our growth and ensures data integrity
